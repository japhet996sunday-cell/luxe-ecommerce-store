Luxe – Premium eCommerce Store

A modern, responsive, production-ready eCommerce website built with HTML5, CSS3, and Vanilla JavaScript.

Luxe is a lightweight frontend eCommerce template designed for developers, freelancers, agencies, startups, and businesses that need a clean, fast, and customizable online store. It requires no frameworks, build tools, or backend services, making it easy to deploy and customize.

Whether you're building a fashion store, electronics shop, furniture store, cosmetics brand, lifestyle shop, or any other retail website, Luxe provides a professional foundation with clean code and an excellent user experience.

Installation

No build tools are required.

Option 1

Download the ZIP file.

Extract it.

Open the project folder.

Launch index.html in your browser.

Option 2

Clone the repository.

git clone https://github.com/japhet996sunday-cell/luxe-ecommerce-store.git

Open the project folder and run it using your preferred local server or open index.html directly in your browser.

Managing Products

Product information is stored inside:

products.json

Each product includes:

- Product name
- Price
- Category
- Description
- Image
- Rating
- Stock status

To add a new product:

1. Open products.json
2. Duplicate an existing product object.
3. Update the product details.
4. Save the file.
5. Refresh the browser.
