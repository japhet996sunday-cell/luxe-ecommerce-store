# LUXE E-Commerce Store

A premium frontend-only eCommerce website built using HTML, CSS, and vanilla JavaScript.

## Features

- Responsive design
- Dynamic product catalog
- Product search and filtering
- Shopping cart
- Wishlist
- Local Storage
- Dark mode
- Product details page
- Checkout UI
- Smooth animations

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
